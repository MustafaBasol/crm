# Internal Quote Detail Page — Content & Actions

A single page to view and manage a quote.

## Header
- Quote number and version (v1, v2…)
- Status pill with timestamps (sent/viewed/accepted/declined/expired)
- Customer and Project/Job links
- Language, currency, exchange rate used

## Body
- Lines table (with optional/bundle/recurring indicators)
- Totals panel (subtotal, discounts, shipping, taxes, grand total, amount‑in‑words)
- Notes & Terms blocks (payment, delivery, warranty)
- Attachments list with preview/download
- Timeline/Audit log (created, edited, sent to X, viewed, accepted/declined with name/IP, reminders, revisions, conversions)

## Actions
- Edit (Draft only)
- Send / Resend / Share public link
- Remind (schedule or send now)
- Revise (creates new version)
- Convert → Invoice / Order (shell)
- Download PDF
- Export (CSV/PDF)
- More: Duplicate, Archive (if supported), Add internal note

## Sidebar (Optional)
- Template selection; Legal footer selection by country/sector
- Permissions hint (who can send/discount)
